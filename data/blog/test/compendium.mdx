## Conditional elements

A line that starts with `[*name]` should be wrapped in a conditional element.

[*name] should be replaced with the name of the checkbox.

A conditional element should is associated with a checkbox that has an id equal to `name`

---

If a checkbox is checked, the conditional element should be shown.

--- 

<Checkbox isIcon id="conditional" spellId={51128} defaultCheck name="Killing Machine" />
[*conditional] This will be visible

---

If a checkbox is not checked, the conditional element should be hidden.

---

<Checkbox isIcon id="conditional-hidden" spellId={51128} name="Killing Machine" />
[*conditional-hidden] This will be hidden

---

You can add a ~ after the * to invert the logic, so the conditional element is shown when the checkbox is not checked.

---

<Checkbox isIcon id="conditional-invert" spellId={51128} defaultCheck name="Killing Machine" />
[*~conditional-invert] This will not be visible

---

<Checkbox isIcon id="conditional-hidden-invert" spellId={51128} name="Killing Machine" />
[*~conditional-hidden-invert] This will be visible

---

A conditional element can be used to show a spell.

---

<Checkbox isIcon id="conditional-spell" spellId={51128} defaultCheck name="Killing Machine" />
[*conditional-spell] !Full Moon!

---

Multiple conditional elements can be controlled by the same checkbox.

---

<Checkbox isIcon id="conditional-multiple" spellId={51128} defaultCheck name="Killing Machine" />

[*conditional-multiple] One !Full Moon! 

[*conditional-multiple] Two !Full Moon!

--- 
Conditional elements can be placed inside a list

<Checkbox isIcon id="conditional-list-one" spellId={51128} defaultCheck name="Killing Machine" />

- [*conditional-list-one] One !Full Moon!
- Two !Full Moon!

--- 

And they should not break the list indentation


<Checkbox isIcon id="conditional-list-two" spellId={51128} defaultCheck name="Killing Machine" />

- One !Full Moon!
- Two !Full Moon!
  - [*conditional-list-two] Three !Full Moon!
