# 기타

## '강한 도트' / Snapshot에 대해

야성 드루이드의 출혈 계열 기술은 현재 내가 보유한 강화 효과의 실시간 상태가 아니라, 기술 시전 시점의 강화 효과 상태를 계속해서 유지하는 특성을 지닌다. 이를 Snapshotting이라고 한다. 예를 들어, !1822|갈퀴 발톱!을 먼저 쓰고 !5217|호랑이의 분노!를 쓰면 그 !1822|갈퀴 발톱!의 지속 피해는 !5217|호랑이의 분노! 효과를 받지 못한다. 반대로 !5217|호랑이의 분노!가 있는 상태에서 !1822|갈퀴 발톱!을 썼다면, !5217|호랑이의 분노! 효과가 만료된 후에도 해당 !1822|갈퀴 발톱!의 지속 피해는 강화된 상태가 유지된다.

갈퀴 발톱은 !5217|호랑이의 분노! 와 은신 상태 (!390772|암습의 일격!: 실제 은신이 아니라 !384667|기습 공격!에 의한 것도 포함) 에 의해 강화된다.  
도려내기는 !5217|호랑이의 분노!와 !319439|피투성이 손길!에 의해 강화된다.  
난타는 !5217|호랑이의 분노! 와 !236068|청명의 전조!에 의해 강화된다.  
(달 바라기 선택 시)달빛섬광은 !5217|호랑이의 분노!에 의해 강화된다.  

이와 같은 Snapshot 상태를 시각화하는 위크오라를 설치할 경우, 각 표기는 다음을 의미한다.  
초록색/100 초과: 지금 해당 기술을 쓰면, 이미 걸려 있던 것보다 강해짐.  
회색/100: 지금 쓸 수 있는 해당 기술의 상태가 이미 걸려 있는 것과 동일함.  
빨간색/100 미만: 지금 해당 기술을 쓰면, 이미 걸려 있던 것보다 약해짐.  

일반적으로 각 출혈 기술을 가능한 한 강력한 상태로 유지하는 것이 좋다. 위의 로테이션 항목은 고려하여 작성된 것이다.

## Pandemic이란?

Pandemic이란, 지속 피해 기술이나 버프 / 디버프를 완전히 만료되기 전에 갱신하면 기존의 남은 지속시간이 이월되는 현상을 의미한다. 그 상한은 전체 지속시간의 30%로, 예를 들어 10초간 지속되는 기술을 2초 남았을 때 리필하면 새 지속시간은 12초가 된다. 하지만 8초가 남았을 때 리필해도 새 것의 지속시간은 13초를 초과하지 못한다.
이 용어는 본래 이 효과를 제공하던 옛날 고통 흑마법사의 특성 이름으로부터 유래했는데, 현재는 거의 모든 지속 피해 기술의 기본 기능으로 탑재되어 있다.


**스마트 휘감는 뿌리**

가능하다면 현재 커서가 올라가 있는(=마우스오버) 대상을 우선적으로 지정하고, 그 외에는 현재 내 캐릭터가 선택한 대상에게 이동 불가 기술인 휘감는 뿌리를 시전하도록 한다.
```
#showtooltip
/cast [@mouseover, harm, nodead] [] 휘감는 뿌리
```

**주시 차단**

가능하다면 현재 주시 대상을 우선적으로 지정하고, 그 외에는 현재 내 캐릭터가 선택한 대상에게 차단기를 시전한다. 주시 대상 지정은 별도로 해야 한다.
```
#showtooltip
/cast [@focus, harm, nodead] [] 두개골 강타
```

**원버튼 적응의 무리**

가능하다면 현재 커서가 올라가 있는 아군 대상을 우선적으로 지정하고, 그 외에는 현재 내 캐릭터가 선택한 대상에게 적응의 무리를 시전한다.
```
#showtooltip
/cast [@mouseover, help, nodead] [] 적응의 무리
```

**전투 부활 / 시너지**

아군 대상에게 커서가 올라가 있다면 해당 아군에게 전투 부활을, 그 외에는 야생의 징표를 시전한다. 해제, 회복, 재생, 정신 자극 등에도 응용하여 단축키 수를 줄일 수 있다.
```
#showtooltip
/cast [@mouseover, help, dead] 환생; 야생의 징표
```
